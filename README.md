## Ihsan Mokhlisse

**OpenShift Consultant & Platform Engineer at [Red Hat](https://redhat.com)** | Paris, France

I help enterprise clients build secure, production-grade container platforms. My focus areas are software supply chain security, GitOps automation, and Kubernetes at scale.

### What I do

- **Supply Chain Security** — Delivered the first [RHTAS](https://www.redhat.com/en/blog/introducing-red-hat-trusted-artifact-signer) (Red Hat Trusted Artifact Signer) implementation in EMEA. Full Sigstore stack: cosign, Rekor, Fulcio, TUF, integrated with Tekton and GitLab CI pipelines.
- **GitOps at Scale** — Design enterprise GitOps operating models with ArgoCD ApplicationSets, Kustomize overlays, and multi-cluster promotion workflows.
- **Open Source Security** — Contributing security hardening features to major open source projects.
